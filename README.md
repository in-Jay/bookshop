# 도서 쇼핑몰 홈페이지
도서를 등록하고 검색할 수 있는 홈페이지를 제작하였습니다.
<br><br>
## 프로젝트 목표
- 도서와 관련된 정보를 이용하여 도서 쇼핑몰 페이지를 개발하고자 하였습니다.
- JSP를 이용하여 백엔드 기능을 구현하고 서비스를 제공하려 했습니다.
- 회원가입, 로그인, 글 작성, 수정, 삭제, 파일 업로드 등의 기능을 구현했습니다.
<br><br>
## 제작기간
- 2024.10.21 ~ 2025.11.15
<br><br>
## 사용기술
- Java
- JavaScript
- JSP
- MySQL
- HTML
- CSS
<br><br>
